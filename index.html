7 Days Down… Survival / Puzzle

▶ Play
Tagline: Seven days underground. No rescue guaranteed.

7 Days… is a 2D side-scrolling survival puzzle game set in a disaster-stricken basement. You manage limited resources and multiple survival stats while responding to scripted day events, threats, and environmental hazards.

Core Features
  • 7-day survival structure
  • Five+ survival stats
  • Interactive environment zones
  • Resource scarcity mechanics
  • Scripted day-based events
Tone / Controls / Status
Tone: Grounded, tense, atmospheric survival.
Controls: Mouse interaction + keyboard movement (if applicable).
Status: Playable survival build.

Akasha: Shatterlayers Card Tactics

▶ Play
Tagline: Shift dimensions. Rewrite reality.

Shatterlayers is a 1–4 player tactical card game where reality shifts across multiple dimensions. Players fight by moving between six reality layers (each with its own rules) and using intuition-based mechanics designed for tense, streamable moments. Dice optional.

Core Features
  • Six reality layers with distinct rule sets
  • Layer-shift mechanics
  • Intuition-based decision moments
  • Dice optional
  • 1–4 player support
Tone / Controls / Status
Tone: Strategic, cerebral, competitive.
Controls: Mouse-based card interaction.
Status: Early tactical prototype.

Deer Xing: Road of Fate Arcade

▶ Play
Tagline: Cross the road. Defy fate.

Deer Xing is a browser-based arcade survival game where you play as a buck navigating dangerous highways. Avoid traffic, grab pickups, and reach safety as levels increase in speed and complexity.

Core Features
  • Classic arcade crossing gameplay
  • Increasing vehicle speed per level
  • Power-up pickups
  • Score & progression tracking
  • Tight keyboard controls
Tone / Controls / Status
Tone: Arcade, reflex-based, minimalist tension.
Controls: Arrow keys or WASD.
Status: Playable arcade build.

Desolation: Tears of the Moon Mgmt / Horror

▶ Play
Tagline: Manage the lab. Control the wolves. Survive the moon.

Desolation: Tears of the Moon is a grim management/survival simulation set in a remote forest lab. You oversee workers, manage resources, and care for werewolves whose power rises and falls with lunar cycles. Day, night, seasons, and moon phases directly affect behavior and survival.

Core Features
  • Werewolf lifecycle & transformation system
  • Moon phase mechanics that alter stats
  • Worker assignment & morale system
  • Breeding & combat system
  • Consequences for neglect or “releasing” staff
  • Dynamic day/night/seasonal cycles
Tone / Controls / Status
Tone: Dark, grim, strategic, psychologically heavy.
Controls: Mouse-driven management interface.
Status: Playable prototype (actively expanding).

Iron Coliseum: Sparks and Blood 2P Fighting

▶ Play
Tagline: Break the armor. End the match.

Iron Coliseum: Sparks and Blood is a simple 2D, local 2-player browser fighting game set in a post-apocalyptic society. Fighters compete in a new combat sport where victory comes from damaging the opponent’s exoskeleton until it fails. Armor breaks first—then the real damage starts.

Core Features
  • Local 2-player fighting (same keyboard)
  • Exoskeleton damage / armor break win condition
  • Fast arcade rounds
  • Post-apocalyptic sport setting
  • Readable 2D combat designed for quick rematches
Tone / Controls / Status
Tone: Brutal, competitive, arcade intensity.
Controls: WASD + Arrow Keys (plus a few others shown on the play screen).
Status: Playable prototype (tuning balance & hit reactions).
Tip: If something doesn’t load online, it’s usually an asset path issue (needs relative paths).
